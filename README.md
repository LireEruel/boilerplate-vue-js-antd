# Vue + JavaScript + Ant Design Vue 템플릿 프로젝트

이 프로젝트는 Vue, JavaScript, 그리고 Ant Design Vue를 활용하여 개발된 템플릿 프로젝트입니다.

## 참고 사항

`src/features/` 프로젝트 내부의 코드는 참고를 위해 등록한 것으로, 제대로 동작하지 않는 코드입니다. 프로젝트 진행시 삭제해주세요.

## 시작하기

### 필요 조건

- Node.js (v14.0 이상 추천)
- Yarn 또는 NPM

### 설치 및 실행 방법

1. 저장소를 클론합니다.
   ```bash
   git clone [저장소 주소]
   ```
2. 필요한 모듈을 다운받습니다.
   ```bash
   yarn
   ```
3. 실행합니다.
   ```bash
   yarn dev
   ```
